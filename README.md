# Throwing Knives Core

Позволяет игрокам бросать ножи.
Плагин может работать как сам, так и быть основой для работы других плагинов (модули вип/шоп)

## Установка

Распаковать архив на сервер.

Конфиг генерируется автоматически при первом запуске плагина в `cfg/sourcemod/ThrowingKnives_Core.cfg`


## Дополнения 

### `TK_Admins_Knives`

Позволяет настроить выдачу ножей для админов


### `TK_NoBlock`

1. NoBlock между игроками: Устанавливает для всех игроков, что позволяет им проходить сквозь друг друга, но не мешает ножам попадать в них.
1. Корректная регистрация попаданий в голову: Использует для точного определения headshot через trace ray:
   - Если расстояние от ножа до головы ≤ 15 единиц — это headshot
   - Если расстояние 15–25 единиц — используется trace ray для уточнения
    > Это решает проблему, когда ножи пролетают через голову
2. Решает проблему затормаживания: Поскольку игроки находятся в группе, они не сталкиваются друг с другом, поэтому попадание ножом в тело не замедляет игрока.
